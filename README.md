# dzikirassalam.com

Static landing page for **Pusat Rawatan Dzikir As-Salam Worldwide**.

- `index.html` — the full self-contained site (inline CSS/JS/SVG).
- `Dockerfile` — serves `index.html` via `nginx:alpine` on port 80.

Deployed via Coolify on the Hostinger VPS. Domains: `dzikirassalam.com` + `www.dzikirassalam.com` (HTTPS via Let's Encrypt).
